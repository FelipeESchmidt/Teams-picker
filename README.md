# What is this App?

App created to pick random teams/group by a list of players/people. You can find it [here](https://teams-picker.herokuapp.com/).

## Technologies used in this Project

- React

- Redux

- Styled-Components

- Font Awesome

The App use Redux to guard all information that can be necessary in another part of the App

## Run App

You can run this App by using [npm](https://www.npmjs.com/) in your own computer, just follow the following steps.

```bash
git clone https://github.com/FelipeESchmidt/Teams-picker.git teams-picker
cd teams-picker
npm i
npm run start
```
